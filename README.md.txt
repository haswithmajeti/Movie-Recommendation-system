# 🎬 Movie Recommendation System

A simple Movie Recommendation System built with **Python**, **Pandas**, and **Streamlit**. This application recommends similar movies based on the selected movie's genre.

---

## 📌 Features

- Recommend movies based on genre
- User-friendly Streamlit interface
- Beginner-friendly Python project
- Easy to customize with more movies

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Streamlit
- Scikit-learn

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/Movie-Recommendation-System.git
```

Install the dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Project

Start the Streamlit application:

```bash
streamlit run app.py
```

---

## 📊 Dataset

The project uses a sample movie dataset containing:

- Movie Title
- Genre

You can replace the dataset with a larger one such as the MovieLens dataset for better recommendations.

---

## 📸 Output

- Select a movie from the dropdown list.
- Click **Recommend**.
- The application displays up to five movies from the same genre.
